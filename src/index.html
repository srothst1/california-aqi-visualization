Open source geospatial tools like CesiumJS can be used to visualize air quality index data and wildfire data in real time. These tools can help people make informed decision on whether it is safe to enjoy outdoor experiences like surfing, hiking, and cycling. The viewer above leverages the object CloudsCollection, which was recently added to the CesiumJS API. Darker clouds indicate poor air quality.
Tweets by LAAirQualityBot
The Google Maps API can also be used to show data of recent wildfires in specific regions. Open source projects such as SoCalAirQuality (https://github.com/ian-r-rose/SoCalAirQuality) are also work highlighting, as they provide a simple way for people to track the air quality index of a given region on Twitter.
